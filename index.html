    Cell Wallet
    Cell Wallet 的定位是为全球用户提供简单可靠的区块链接入服务，作为区块链世界和用户的中间层，Cell Wallet将本着至臻的设计理念，给用户打造安全无忧、简约而不简单的区块链接入体验。

  • 02
    Cell 质押
    Consensus Cell Network 选择了 eCell Token 作为网络主要的流通、治理、激励凭证，eCell Token 的发行存在目标通胀机制，将为 Consensus Cell Network 提供合理的经济及激励模型。基于 eCell 的去中心化质押委托机制，我们构筑了Consensus 节点（共识节点）体系，其中 eCell的去中心化质押委托机制将通过 Cell 质押实现。
  • 03
    Cell 闪兑
    Cell 闪兑的做市商机制是做市商双边竞价机制，对于 Cell 闪兑内支持的交易货币对，做市商可根据其持仓情况，选择相关货币对进行双边报价，Consensus Cell Network 将通过做市商招募、合作计划，吸收来自于全球的做市商，汇集他们的流动性与报价，对用户来说，Cell 闪兑将在交易所前台页面自动为用户选择最优的交易报价，提高市场总体的流动性。

业务板块

Consensus Cell Network 业务包含6大领域

技术研究

公链基础设施技术开发研究。公链是区块链的底层协议，是区块链世界的“操作系统”。

应用开发

数字资产钱包（Cell Wallet）应用开发。让 Cell Wallet 成为用户通往新世界的大门，致力于为全球用户提供安全可靠的数字资产管理及区块链内容平台服务。

风险投资

精选优质投资标的，以风险投资的形式实现行业资源的整合，推动区块链产业发展。

企业服务

可为客户提供定制化的开发服务，提供包括公链基础设施、去中心化应用、垂直应用在内的多层解决方案；提供覆盖需求调研、需求分析、产品设计、产品开发、产品部署、售后支持在内的全流程服务。

传媒教育

Consensus Cell Network 将以投资整合的形式，汇集全球开发者社区，通过社区的形式更好地去承载区块链技术的知识体系，以分布于全球的社区为基点，面向社会公众，由点及面，将开发者社区的用户边界拓宽，使其成为履行教育职能的良好媒介。

公益慈善

Consensus Cell Network 将依托相关技术应用储备、推出 Consensus Cell Network 公益慈善链并成立 Consensus Cell Network 慈善公益平台。

组织治理

依托于 Cell 质押建立 Consensus 节点体系，Consensus 节点是支撑Consensus Cell Network 发展的基石力量。Consensus 节点也担任着 Consensus Cell Network 的主要治理职能，具体的治理机制包括 Consensus Cell Network 的组织治理以及去中心化应用的治理两个层面。
