共识节点

Cell Wallet 的定位是为全球用户提供简单可靠的区块链接入服务
eCell 是建筑经济模型的基础通证，同时我们建立了节点机制，一方面让基于节点进行 Consensus Cell Network 组织的去中心治理成为可能，一方面我们将通过适当的激励机制，让节点成为业务场景推广过程的内生动力。未来 Consensus 节点成员将由 Consensus Cell Network 网络成员通过智能合约投票选举产生，Consensus 节点是 Consensus Cell Network 组织的基本治理单元与 Consensus Cell Network 生态的基本业务推广单元。
它具有以下作用

01 节点作为基本的治理单元
Consensus 节点将与 Consensus Cell Network 去中心化治理层内的智能合约构成去中心化治理架构。Consensus 节点共同组成了“理事会”性质的职能组织，Consensus 节点将依托于配套的智能合约，对 Consensus Cell Network 经营的重大事项进行群体性决策，Consensus 节点可发起提案，全体 Consensus 节点通过股份加权投票机制对所有经营提案进行决策；对于未成为节点的一般用户而言，用户可通过将所持有的 eCell 委托于某些节点，节点将代表与节点发生质押委托关系的所有用户行使治理权力；在节点处质押的 eCell 的数量多寡将影响节点参与决策时的投票权重；用户可看到其质押节点的决策情况，倘若某些节点的决策与用户预期相背，可能会导致质押用户的流失从而降低节点的投票权重，这也一定程度对节点的治理行为产生约束，降低不当决策出现的可能。

02节点作为基本的推广单元
Consensus 节点将与 Consensus Cell Network 生态层内的一部分业务板块形治理机制50 Consensus Cell Network成对接与协同，建立推广激励机制。推广激励机制将允许 Consensus 节点通过推广生态层内的应用项目获得对应的奖励。Consensus 节点将成为推进生态层发展的强有力内生动能，形成技术成长—应用推广—技术成熟—应用普及—大规模应用的积极正循环，对此我们建立了配套的业务激励机制：
（1）每个节点都将拥有配套的推广链接
（2）用户通过推广链接可访问相关应用场景
（3）用户通过节点的推广链接进行相关业务操作，节点将获得相应奖励
